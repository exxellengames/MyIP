# MyIP

A utility for viewing internal and external IP addresses, testing the network and checking connectivity.

## Screenshot
![myip](https://github.com/user-attachments/assets/553ec482-525e-4f3c-99ee-5c2648759893)


## Features
- Displays the local (internal) IP address.
- Displays the external (public) IP address
- Displays LAN settings
- Testing your network connection
- Ping utility
- Simple GUI
- Windows only
