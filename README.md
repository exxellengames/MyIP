# MyIP

Utility for viewing internal and external IP addresses, network testing, and ping.

## Screenshot
![myip](https://github.com/user-attachments/assets/b6b52032-29b9-44ca-ab85-6249a2c6d889)

## Features
- Displays local (internal) IP address
- Displays external IP address
- Network connectivity testing
- Ping utility
- Simple GUI
- Windows only
